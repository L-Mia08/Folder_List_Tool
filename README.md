# Folder List Tool_v0.3  
  
  
このスクリプトはリストでファイルパスを管理、エクスプローラーを起動できるスクリプトです。  
難しい機能はないため、感覚で使えれると思います。

  
《ダウンロード》  
CodeをクリックしてDownload ZIPからダウンロードできます。  

《アップデート》  
新しい「Folder_List_Tool」フォルダーと「Folder_List_Tool_im.py」のファイルを上書き保存するだけです。  　

《主な機能》  
・リストから選択してエクスプローラーを起動  
&nbsp; +ファイルを選択して追加  
&nbsp; +ファイルパスを手動して追加  
・作業フォルダの作成(fbx, obj, psd, reference, ssp, zb)  
・リストパスをiniファイルで保存して別シーンでも使用可能  
etc…  

 
《導入方法》  
英語版をお使いの方は「Folder_List_Tool」のフォルダーと「Folder_List_Tool_im.py」を  
"C:\Users\{名前}\Documents\maya\2025\scripts"に入れてください。  
ファイル名を指定して実行で
```
%USERPROFILE%\Documents\maya\2025\scripts
```
を入力すると一発でフォルダーが開きます。  
Mayaのスプリクトエディターを起動して「Folder_List_Tool_im.py」実行するとGUIが表示されます。  
  
日本語版の方は"C:\Users\{名前}\Documents\maya\2025\ja_JP\scripts"に入れると起動できるはずです。  
動作確認はMaya2023とMaya2025です。macOSは未確認です。  
  
  
《使い方》  
起動するとリスト表「user_path_data_list.ini」を読み込みます。  
ない場合はフォルダとファイルが作成されます。
